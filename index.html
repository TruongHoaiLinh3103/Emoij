Khái niệm

Emoji Unicode là những đoạn mã icon nằm trong bộ Unicode là bộ mã chuẩn quốc tế được thiết kế với mục đích làm mã duy nhất dùng chung cho tất cả các loại ngôn ngử nhờ đó tất cả emoji code trong Unicode sẽ được hiển thị trên tất cả các hệ thống nào mà không lo gặp lỗi. 

Sử dụng Emoji Unicode trong HTML

Để sử dụng Emoji Unicode trong HTML rất đơn giản, bạn chỉ cần thêm đoạn mã &# cộng với mã Des trong bộ Unicode tương ứng thì hệ thống sẽ tự chuyển đổi sang icon tương ứng với code đó.
